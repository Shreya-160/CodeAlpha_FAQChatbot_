# CodeAlpha FAQ Chatbot

## Project Description
This project is an FAQ Chatbot developed using Python and Machine Learning techniques.

## Technologies Used
- Python
- Scikit-Learn
- TF-IDF Vectorizer
- Cosine Similarity

## Features
- Answers predefined questions
- Natural Language Processing
- Similarity-based response generation

## Installation

```bash
pip install scikit-learn
```

## Run

```bash
python chatbot.py
```
